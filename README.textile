h1. Guiki (Graphical Wiki)

h2. AUTHOR: Matt Lankford

h2. WEBSITE: www.guiki.com

h2. LICENSE: GPL

h2. WHAT:

Guiki is my attempt to scratch my own itch for a simple content management system.

h2. WHY:

It is very easy to install and use and does not have a lot of the fluff other wiki systems and content management systems include.

h2. REQUIREMENTS:

A webserver and PHP 5.x and above

h2. FEATURES:

Very simple
Easy to create websites for normal people
Easy to add content
Great for intranets / documentation of company functions
Search function

h2. INSTALL:

Using GIT: git clone git://github.com/lankford/guiki.git

or just download from github.com/lankford/guiki, and unzip

h2. SETUP:

Change the user and pass and site name in the index.php
